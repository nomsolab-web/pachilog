# パチログ (PachiLog) — Design System

## コンセプト
パチスロ・パチンコ系YouTuberの「登録者数・再生数の推移」を定点観測するデータトラッカー。
攻略・期待値は一切扱わないので、ギャンブル感を出さず「分析ダッシュボード」寄りのクリーンなトーンにする。
ネオン的な差し色はデータの上下（急上昇/急降下）を直感的に伝えるために使う。

## カラー
ダークベース + ネオンアクセントの2トーン構成。CSS変数で管理。

```css
--bg: #0B0E14;           /* ページ背景 */
--surface: #141924;      /* カード背景 */
--surface-alt: #1B2230;  /* ホバー・区切り */
--border: #262E3D;

--text-primary: #F2F5F9;
--text-secondary: #8B96A8;

--accent-rise: #34F5A3;  /* 急上昇・プラス値 */
--accent-fall: #FF5C7A;  /* 急降下・マイナス値 */
--accent-gold: #FFC857;  /* マイルストーン・強調・パチスロらしさ */
--accent-blue: #4C8DFF;  /* リンク・情報系 */
```

- 背景はグラデーションメッシュ（`--bg`から少し明るい紺への放射状グラデーション）を薄く敷く
- カードは角丸12px、`--border`の1px枠、ホバーで`--surface-alt`にトランジション

## タイポグラフィ
- 本文・UI：**Noto Sans JP**（日本語の可読性優先）
- 数値・統計（登録者数、増減率など）：**Poppins**（Bold/SemiBold）— 数字だけ欧文フォントにして「データ感」を強調
- 見出し階層：H1 32px/Bold、H2 22px/SemiBold、本文14-16px/Regular、行間はゆったり(1.6)

## レイアウト
- 12カラムグリッド、最大幅1200px中央寄せ
- トップページ：ヒーロー（急上昇TOP3を大きく）→ ランキングタブ（急上昇/急降下/総合）→ チャンネル一覧グリッド
- チャンネル詳細ページ：左にプロフィール＋投票UI、右に推移グラフ（登録者数/再生数タブ切替）
- 密度は中程度：情報量は多いが余白でグルーピングして圧迫感を出さない

## コンポーネント
- **ランキングカード**：サムネ＋チャンネル名＋現在値＋増減バッジ（▲緑 / ▼赤、%表示）
- **グラフ**：Rechartsのエリアチャート、線は`--accent-blue`、上昇区間は`--accent-rise`のグラデーション塗り
- **投票UI**：好き/嫌い/わからないの3ボタン、投票後に円グラフで割合表示
- **Xシェアボタン**：`--accent-gold`背景、グラフのスナップショット画像を生成してX intent URLに渡す
- **マイルストーンバッジ**：金色のリボン風、"10万人突破 🎉"のようなテキスト（絵文字は最小限、lucide-reactアイコン優先）

## モーション
- ページロード時にランキングカードを上から下へ100msずつずらしてフェードイン
- 数値カウントアップアニメーション（登録者数が0→実数へアニメ）
- カード全体のホバーは薄い浮き上がり(shadow + translateY(-2px))のみ、過剰な演出はしない

## アンチパターン（避ける）
- 紫グラデーション、量産型の角丸カードグリッドの単調な羅列
- ギャンブル台のような点滅・けばけばしい配色
- Inter/Space Grotesk/Robotoの多用（本プロジェクトはNoto Sans JP + Poppins固定）
